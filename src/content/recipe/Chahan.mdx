## 材料

### 食材

| 材料         | 分量       |
| ------------ | ---------- |
| ご飯         | 200g       |
| 卵           | 2個        |
| チャーシュー | 100g       |
| 長ネギ       | 好きなだけ |

### 味付け

| 材料   | 分量      |
| ------ | --------- |
| ラード | 40g       |
| 塩     | 小さじ1/2 |
| 味の素 | 小さじ1/2 |
| 胡椒   | 3振り     |

## 手順

1. チャーシューと長ネギを細かく切る
2. フライパンにラードを入れ、熱する
3. ラードが十分に温まったら、溶いた卵を一気に入れる
4. すぐにご飯を入れてよく混ぜる
5. チャーシューを加えてさらに混ぜる
6. 塩、味の素、胡椒を加える
7. 最後に長ネギを加えて軽く混ぜる
